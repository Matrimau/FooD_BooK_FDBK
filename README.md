# FooD_BooK
This repository is only for my university study project

Данный репозиторий будет представлять из себя подробный (или нет) личный дневник питания и создан исключительно в образовательных целях

## Суть
Cуть данного репозитория заключается в следующем: 
- Записи потребляемых продуктов
- Подсчёте калорий
- БЖУ (белки, жиры, углероды)
- Отчёт

## Сущности
| **Продукт** | **Калории** | **БЖУ** |
| --- | --- | --- |
| Помело | 32–38 ккал на 100 г | 0,6–0,8 г белков, 0,1–0,2 г жиров и 6,7–9,6 г углеводов |
| Пиво | 30–35 ккал на 100 г | 0,46 г белков, 0 г жиров, 3,55 г углеводов |
| Хинкали | 210 ккал на 100 г | 7,5 г белков, 7,7 г жиров, 26 г углеводов |
| Хачапури мегрельский | 302 ккал на 100 г | 12 г белков, 17,6 г жиров, 23 г углеводов |
